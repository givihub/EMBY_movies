# 🎬 Emby Media Renamer

Набор Python-скриптов для автоматического переименования фильмов и сериалов по данным из NFO-файлов, созданных Emby. Приводит структуру папок и имена файлов к стандарту, который Emby распознаёт наилучшим образом.

## Требования

- Python 3.10+
- Стандартные библиотеки (xml, pathlib, csv — ничего устанавливать не нужно)
- Emby Server должен быть **остановлен** перед запуском с `--apply`

## rename_movies.py — Фильмы

### Запуск

```powershell
python rename_movies.py "D:\Media\Movies" --apply
```

Без флага `--apply` скрипт не запустится — это защита от случайного запуска.

### Что делает

Скрипт анализирует папку с фильмами, читает NFO-файлы и переименовывает папки и файлы по единому стандарту. После применения запрашивает подтверждение `[да/нет]`.

### Типы папок

| Тип | Описание | Действие |
|-----|----------|----------|
| 📄 Фильм без папки | `Фильм.mkv` + `Фильм.nfo` в корне | Создаётся папка, файлы перемещаются |
| 📁 Фильм в папке | Одна папка — один фильм | Папка и файлы переименовываются |
| 🗂️ Сборник с `_` | `_Форсаж/`, `_Star Wars/` | Обрабатываются подпапки внутри |
| 🗂️ Сборник без `_` | Несколько фильмов в одной папке | Автоматически добавляется `_`, обрабатываются подпапки |

### Что переименовывается

- ✅ Папка фильма
- ✅ Видеофайл (`.mkv`, `.mp4`, `.avi` и др.)
- ✅ NFO-файл + обновляются `lockdata` и `lockedfields` внутри
- ✅ Субтитры (`.srt`, `.ass`, `.ssa`, `.sub` и др.)

### Что **не** переименовывается (только перемещается в папку)

- `poster.jpg`, `poster.png`
- `fanart.jpg`, `fanart1.jpg` ... `fanart8.jpg`
- `landscape.jpg`, `clearlogo.png`, `clearart.png`
- `banner.jpg`, `thumb.jpg`, `disc.jpg`

### Лог

После применения создаётся файл `rename_log_YYYYMMDD_HHMMSS.csv` в папке с фильмами со списком всех операций и их статусом.

## rename_shows.py — Сериалы

### Запуск

```powershell
# Один сериал
python rename_shows.py "M:\Video\Serials\Silicon Valley" --apply

# Все сериалы в папке
python rename_shows.py "M:\Video\Serials" --all --apply
```

### Что делает

Читает `tvshow.nfo` для имени сериала и `episodedetails` NFO каждого эпизода для номеров сезона/серии. Создаёт правильную структуру папок и переименовывает файлы.

### Поддерживаемые структуры эпизодов

Скрипт автоматически распознаёт все варианты:

| Исходная структура | Результат |
|-------------------|-----------|
| `01/01.mkv` (папки-числа) | `Season 01/Сериал 2014 S01E01.mkv` |
| `Сезон 1/01.mkv` | `Season 01/Сериал 2014 S01E01.mkv` |
| `Season 01/01.mkv` | `Season 01/Сериал 2014 S01E01.mkv` |
| `S01E01.mkv` прямо в папке сезона | `Season 01/Сериал 2014 S01E01.mkv` |
| Все эпизоды в одной папке | `Season 01/...`, `Season 02/...` |

### Итоговая структура

```
Silicon Valley (Кремниевая долина) 2014/
    tvshow.nfo
    poster.jpg
    Season 01/
        Silicon Valley (Кремниевая долина) 2014 S01E01.mkv
        Silicon Valley (Кремниевая долина) 2014 S01E01.nfo
        Silicon Valley (Кремниевая долина) 2014 S01E01-thumb.jpg
        ...
    Season 02/
        ...
```

## Логика именования

### Фильмы

| Условие | Формат | Пример |
|---------|--------|--------|
| Только Россия/СССР в `<country>` И названия совпадают | `Название год` | `Грешники 2025` |
| `<originaltitle>` отличается от `<title>` | `OriginalTitle (Title) год` | `Sinners (Грешники) 2025` |
| Нет `<country>`, названия совпадают | `Название год` (автоматически) | `12 стульев 1971` |
| Нет `<country>`, названия разные | `OriginalTitle (Title) год` (автоматически) | `Gandhi (Ганди) 1982` |
| Россия + другие страны, названия разные | `OriginalTitle (Title) год` | `Anna (Анна) 2019` |
| Неоднозначность (нет страны, названия разные) | Спрашивает пользователя | — |

### Конфликт имён

Если два фильма получают одинаковое имя — к дублю добавляется `(1)`, `(2)` и т.д.

### Обновление NFO

В каждый обработанный NFO добавляется или обновляется:
```xml
<lockdata>true</lockdata>
<lockedfields>Name|OriginalTitle|SortName|Overview|Genres|Cast</lockedfields>
```

### Одиночный фильм

```
Movies/

        Sinners (Грешники) 2025.mkv
        Sinners (Грешники) 2025.nfo
        poster.jpg
        fanart1.jpg

```

### Сборник

```
Movies/
    _The Fast & the Furious (Форсаж)/
        The Fast & the Furious (Форсаж) 2001/
            The Fast & the Furious (Форсаж) 2001.mkv
            The Fast & the Furious (Форсаж) 2001.nfo
            poster.jpg
        2 Fast 2 Furious (Двойной форсаж) 2003/
            2 Fast 2 Furious (Двойной форсаж) 2003.mkv
            ...
```

## Известные ограничения

- **Emby должен быть остановлен** перед запуском — иначе NFO-файлы заблокированы и патч не применится (видео и папки всё равно переименуются).
- Папки без NFO пропускаются с предупреждением.
- Сборники с фильмами вперемешку (без подпапок) обрабатываются — для каждого NFO создаётся отдельная подпапка автоматически.
- `<country>` отсутствует во многих NFO — скрипт определяет российский/зарубежный по разнице названий.
